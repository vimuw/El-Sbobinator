# Code of Conduct

## Our Commitment

We want El Sbobinator to be a welcoming, respectful, and constructive project for everyone.

Contributors, maintainers, and community members are expected to participate in a way that helps keep discussions collaborative, professional, and inclusive.

## Expected Behavior

Examples of behavior that contribute to a positive environment include:

- Being respectful and considerate in communication
- Giving and receiving feedback in good faith
- Focusing on what is best for the project
- Being patient with contributors at different experience levels
- Accepting responsibility and learning from mistakes

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, insults, or personal attacks
- Trolling, intimidation, or deliberately disruptive behavior
- Discriminatory or exclusionary language or conduct
- Publishing private information without permission
- Repeated bad-faith arguments or hostility in issues, pull requests, or discussions

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues
- Pull requests
- Code review discussions
- Project-related chats or other public interactions tied to this repository

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

If you experience or witness unacceptable behavior, contact the project maintainers through GitHub private channels where possible rather than escalating in public.

## Consequences

Maintainers may take any action they deem appropriate, including:

- Requesting changes in behavior
- Removing inappropriate content
- Temporarily limiting participation
- Banning repeat or severe offenders from project spaces
